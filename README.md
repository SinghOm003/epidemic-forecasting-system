# epidemic-forecasting-system
Developed a data-driven epidemic forecasting application using ARIMA and LSTM time-series models to predict infection trends from historical data, supported by exploratory analysis and visualizations for early and informed decision-making.
# Epidemic Forecasting System

This project predicts epidemic trends using machine learning techniques.

- Pandas
- Matplotlib
- Jupyter Notebook

## Features
- Data preprocessing
- Time-series visualization
- Prediction modeling
